## 프로젝트 구조

```
/
 ├─ frontend/          # 프론트엔드 애플리케이션
 │   ├─ package.json
 │   └─ ...
 ├─ backend/           # 백엔드 애플리케이션
 │   ├─ pyproject.toml
 │   └─ ...
 ├─ docker-compose.yml # Docker Compose 설정
 └─ README.md
```

## 시작하기

### 개발 환경 설정

#### 프론트엔드
```bash
cd frontend
npm install
npm run dev
```

#### 백엔드
```bash
cd backend
pip install -e .
```

### Docker를 사용한 실행
```bash
docker-compose up
```
